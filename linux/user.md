# User

```
$ whoami
```

```
$ id
```

```
$ groups
```

### Add User

```shell
$ useradd [username]
```

```shell
$ userdel [username]
```

### Modify User

```shell
$ usermod [username]
```

```shell
$ passwd [username]
```

### Add User to Group

```shell
$ usermod -a -G [groupname] [username]
```
