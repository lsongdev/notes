# 数学

### EMC

$E = mc^2$

### Fibonacci

$F_n = F_{n-1} + F_{n-2}$

```
fn = n => n < 2 ? n : fn(n-1) + fn(n-2)
```

### Golden ratio

$\phi = \frac{1 + \sqrt{5}}{2}$


### Euler's formula

$e^{i\pi} + 1 = 0$

### Fourier series

$ f(t) = a_0 + \sum_{n=1}^{\infty} (a_n \cos(nt) + b_n \sin(nt))$

### Fast Fourier Transform

$X_k = \sum_{n=0}^{N-1} x_n e^{-i 2 \pi k n / N}$
